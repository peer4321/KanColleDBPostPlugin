# KanColleDBPostPlugin

通信データを艦これ統計データベースへ送信する KanColleViewer 用プラグインのようなものです。

### インストール

* `DBPostPlugin.dll` を KanColleViewer の `Plugins` ディレクトリに移動してください。

### 使い方

* ツール→DBPostタブの艦これ統計データベースへデータを送信しますを有効にし、
アクセスキーを入力することで使用できます。
* 変更した設定は自動的に保存されます。
